# CareerConnect Job Portal

CareerConnect Job Portal is a web-based application built using Spring Boot and MySQL. The project allows users to register, log in, view job listings, manage users, and track job applications through a simple and responsive interface.

### Features

* User Registration
* User Login
* Job Listings
* User Management
* Application Tracking
* Dashboard Statistics

### Tech Stack

* Java
* Spring Boot
* Spring MVC
* Spring Data JPA
* Spring Security
* Thymeleaf
* MySQL
* Hibernate
* Maven
* HTML
* CSS
* Git
* GitHub
